# Fake News Detection

## Project Description
This project detects whether a news article is **Fake** or **Real** using **Machine Learning** and **Natural Language Processing (NLP)** techniques. The system analyzes the textual content of news articles and classifies them accurately. It also generates a **confusion matrix** to evaluate model performance and reliability.

---

## Features
- Classifies news articles as Fake or Real
- Preprocesses textual data for better analysis
- Generates a confusion matrix to show model accuracy
- Easy-to-run Python implementation in a single file
